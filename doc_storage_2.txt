Technical README Component: The "Ken Burns" CSS/JS Logic
To achieve the "Google/Apple" rendering of the Ken Burns effect in your web sandbox, use this logic:
css
/* World-Class Ken Burns Animation for Documentaries */
@keyframes kenburns-effect {
  0% {
    transform: scale(1.0) translate(0, 0);
  }
  100% {
    transform: scale(1.2) translate(-20px, -10px); /* Subtle zoom & pan */
  }
}


.documentary-canvas {
  width: 100%;
  height: 100vh;
  overflow: hidden;
  background-color: #000;
}


.documentary-image {
  width: 100%;
  height: 100%;
  object-fit: cover;
  animation: kenburns-effect 12s ease-in-out infinite alternate;
}
